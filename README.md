## Getting started

#### Requirements

- [Emacs](https://www.gnu.org/software/emacs/) v26.x or higher

#### Setup

1. Clone this repo:

```console
$ git clone git@github.com:otype/emacs.d.git ~/.emacs.d
```

2. Start Emacs:

```console
$ emacs
```

This will automatically install all packages on startup.
